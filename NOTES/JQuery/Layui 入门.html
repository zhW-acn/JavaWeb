一、核心内容

1、基本使用

  1. 引入

需要分别引入 layui.css 和 layui.js 文件（下载本地引入、CDN引入、npm 下载）

下载至本地：官网首页点击“直接下载”接口

CDN引入

<!-- 引入 layui.css -->
<link href="//unpkg.com/layui@2.7.6/dist/css/layui.css" rel="stylesheet">
 
<!-- 引入 layui.js -->
<script src="//unpkg.com/layui@2.7.6/dist/layui.js"></script>

npm 下载：npm i layui

官网提供的代码包经过了自动化构建，更适合用于生产环境，目录结构如下：

  layui/
    ├── css
    │   ├── layui.css    # 核心样式库
    │   └── modules
    │       ├── code.css

    │               ├── loading-0.gif

    └── layui.js    # 核心样式库
  1. 加载所需模块: layui.use([mods], callback)

mods里面必须是一个合法的模块名，不能包含目录。callback即回调函数。如下：

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1" />
    <title>开始使用 layui</title>
    <!-- 引入 layui.css -->
    <link href="//unpkg.com/layui@2.7.6/dist/css/layui.css" rel="stylesheet" />
  </head>
  <body>
    <!-- 你的 HTML 代码 -->

    <!-- 引入 layui.js -->
    <script src="//unpkg.com/layui@2.7.6/dist/layui.js"></script>
    <script>
      layui.use(["layer", "form"], function () {
        var layer = layui.layer;
        var form = layui.form;

        layer.msg("Hello World");
      });
    </script>
  </body>
</html>
